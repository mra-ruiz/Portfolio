Scratch

The first week, we worked with Scratch, a drag and drop programming language

Project 1: Pitfall

This team project was created from given basket items. That is, we were given a few blocks that we had to use to create whatever we wanted, so we make an animation of a man that walks across water to the beat of a drum, only to bounce back and fall in the pit! View the program here!

Project 2: Square and Hexagon

In this program, I programmed the cat to be able to draw a square and a hexagon when told to by the user. View the program here!

Project 3: Paddle Ball

For this program, the user moves the platform left and right using the arrow keys. The purpose of this game is to make sure the ball does not touch the bottom of the screen. View the program here!

Project 4: Pig Game

This is a two player game. The purpose of this game is to become the first player to reach a score of 100!View the program here!
